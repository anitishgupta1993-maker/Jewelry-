# 💎 Noor Jewels - Luxury Indian Jewelry E-Commerce Website

A beautiful, modern e-commerce website for authentic Indian jewelry featuring bridal collections, temple jewelry, Kundan & Polki designs, and more.

## ✨ Features

- 🛍️ **Complete Shopping Cart System** - Add, remove, and manage items with quantity control
- 📱 **Fully Responsive Design** - Mobile, tablet, and desktop optimized
- 💰 **Real-time Pricing** - Automatic tax calculation and total updates
- 🎨 **Luxury UI Design** - Gold (#c9a46c) and brown (#8b5e3c) premium aesthetic
- 📸 **High-Quality Images** - Professional product and collection photography
- 📧 **Contact Form** - Customer inquiry and feedback system
- ⭐ **Testimonials Section** - Customer reviews and ratings
- 🔍 **SEO Optimized** - Proper meta tags and semantic HTML

## 🚀 Quick Start

### Installation
```bash
npm install
```

### Development Server
```bash
npm start
```
Open [http://localhost:3000](http://localhost:3000) in your browser.

### Production Build
```bash
npm run build
```

## 📦 Tech Stack
